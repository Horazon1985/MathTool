<html><head></head><body><b>&Uuml;BER MATHTOOL:</b></body></html>

MathTool ist eine Eingabekonsole, die der Verarbeitung und der Auswertung mathematischer Ausdr&uuml;cke dient.


Mathematische Ausdr&uuml;cke:

MathTool versteht mathematische Ausdr&uuml;cke. Hierbei werden die vier Grundrechenarten mit +, -, * und / angegeben, Potenzen durch ^. Mathematische Funktionen tragen ihre &uuml;blichen Namen, z.B. sin(...), exp(...), ln(...), arctan(...) etc. <br>
Wichtig: Multiplikationen m&uuml;ssen stets durch ein * angegeben werden. MathTool versteht den Ausdruck 2*x, dagegen jedoch 2x nicht und gibt dementsprechend einen Fehler aus.<br>
G&uuml;ltige Ausdrücke sehen dann etwa wie folgt aus:

3*x + y^5, -a + 6*exp(2 + cos(z^4)), ...

Ein eingegebener mathematischer Ausdruck wird von MathTool sofort in vereinfachter Form ausgegeben, sofern dies m&ouml;glich ist.



Mathematische Operatoren:

MathTool kennt die folgenden Operatoren:

<b>OPERATOR diff(FUNCTION, VAR):</b> erechnet die partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.

<b>OPERATOR diff(FUNCTION, VAR, k):</b> Berechnet die k-te partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.

<b>OPERATOR div(FUNCTION):</b> Berechnet die Divergenz der Funktion FUNCTION.

<b>OPERATOR gcd(a_1, ..., a_k):</b> Berechnet den gr&ouml;&szlig;ten gemeinsamen Teiler der ganzen Zahlen a_1, ..., a_k.

<b>OPERATOR int(FUNCTION, x_1, x_2):</b> Berechnet das bestimmte Integral &uuml;ber die Funktion FUNCTION von x_1 bis x_2.

<b>OPERATOR laplace(FUNCTION):</b> Berechnet den Laplace-Operator der Funktion FUNCTION.

<b>OPERATOR lcm(a_1, ..., a_k):</b> Berechnet das kleinste gemeinsame Vielfache der ganzen Zahlen a_1, ..., a_k.

<b>OPERATOR prod(EXPRESSION, VAR, k_1, k_2):</b> Berechnet das Produkt f&uuml;r den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.

<b>OPERATOR sum(EXPRESSION, VAR, k_1, k_2):</b> Berechnet die Summe f&uuml;r den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.

<b>OPERATOR taylor(FUNCTION VAR, x_0, k):</b> Berechnet das Taylorpolynom k-ten Grades von der Funktion FUNCTION bez&uuml;glich der Variablen VAR an der Stelle VAR = x_0.

Operatoren können wiederum in weiteren mathematischen Ausdrücken vorkommen und somit weiterverarbeitet werden. So wird beispielsweise der Ausdruck

x^2*diff(y^5,y,3) + 6

vereinfacht zu

60*x^2*y^2+6.



Konsolenbefehle:

MathTool kennt die folgenden Befehle:

BEFEHL <b>def(VAR = VALUE):</b> Weist einer g&uuml;ltigen Variable VAR den Wert VALUE zu.

BEFEHL <b>def(FUNCTION = EXPRESSION):</b> Definiert eine Funktion durch den Ausdruck EXPRESSION. 

BEFEHL <b>defvars():</b> Gibt alle Variablen aus, denen ein Wert zugewiesen wurde.

BEFEHL <b>latex(EXPRESSION):</b> Generiert den Latex-Code für den mathematischen Ausdruck EXPRESSION.

BEFEHL <b>plot(FUNCTION(VAR), x_1, x_2):</b> Zeichnet den Graphen der Funktion FUNCTION im Intervall von x_1 <= VAR <= x_2.
  </b>
BEFEHL <b>plot(FUNCTION(VAR_1, VAR_2), x_1, x_2, y_1, y_2):</b> Zeichnet den Graphen der Funktion FUNCTION von zwei Variablen im Rechteck x_1 <= VAR_1 <= x_2, y_1 <= VAR_2 <= y_2.
Wichtig: Die Grenzen x_1, x_2, y_1, y_2 sollen so angegeben werden, dass VAR_1, VAR_2 die alphabetische Reihenfolge der Variablen ist.

BEFEHL <b>undef(VAR_1, ..., VAR_n):</b> Gibt die variablen VAR_1, ..., VAR_k wieder frei.
