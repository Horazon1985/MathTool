Mathematische Ausdrücke:

MathTool versteht mathematische Ausdrücke. Hierbei werden die vier Grundrechenarten mit +, -, * und / angegeben, Potenzen durch ^. Mathematische Funktionen tragen ihre üblichen Namen, z.B. sin(...), exp(...), ln(...), arctan(...) etc. 
Wichtig: Multiplikationen müssen stets durch ein * angegeben werden. MathTool versteht 2*x, dagegen jedoch 2x nicht und gibt dementsprechend einen Fehler aus.



Mathematische Operatoren:

MathTool kennt die folgenden Operatoren:

OPERATOR diff(FUNCTION, VAR): Berechnet die partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.

OPERATOR diff(FUNCTION, VAR, k): Berechnet die k-te partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.

OPERATOR div(FUNCTION): Berechnet die Divergenz der Funktion FUNCTION.

OPERATOR gcd(a_1, ..., a_k): Berechnet den größten gemeinsamen Teiler der ganzen Zahlen a_1, ..., a_k.

OPERATOR int(FUNCTION, x_1, x_2): Berechnet das bestimmte Integral über die Funktion FUNCTION von x_1 bis x_2.

OPERATOR laplace(FUNCTION): Berechnet den Laplace-operator der Funktion FUNCTION.

OPERATOR lcm(a_1, ..., a_k): Berechnet das kleinste gemeinsame Vielfache der ganzen Zahlen a_1, ..., a_k.

OPERATOR prod(EXPRESSION, VAR, k_1, k_2): Berechnet das Produkt für den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.

OPERATOR sum(EXPRESSION, VAR, k_1, k_2): Berechnet die Summe für den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.

OPERATOR taylorFUNCTION VAR, x_0, k): Berechnet das Taylorpolynom k-ten Grades von der Funktion FUNCTION bezüglich der Variablen VAR an der Stelle VAR = x_0.



Konsolenbefehle:

MathTool kennt die folgenden Befehle:

BEFEHL def(VAR = VALUE): Weist einer gültigen Variable VAR den Wert VALUE zu.

BEFEHL def(FUNCTION = EXPRESSION): Definiert eine Funktion durch den Ausdruck EXPRESSION.

BEFEHL defvars(): Gibt alle Variablen aus, denen ein Wert zugewiesen wurde.

BEFEHL plot(FUNCTION, x_1, x_2): Zeichnet den Graphen der Funktion FUNCTION im Intervall von x_1 bis x_2.
  
BEFEHL plot(FUNCTION(VAR_1, VAR_2), x_1, x_2, y_1, y_2): Zeichnet den Graphen der Funktion FUNCTION von zwei Variablen im Rechteck x_1 <= VAR_1 <= x_2, y_1 <= VAR_2 <= y_2.
Wichtig: Die Grenzen x_1, x_2, y_1, y_2 sollen so angegeben werden, dass VAR_1, VAR_2 die alphabetische Reihenfolge der Variablen ist.

BEFEHL undef(VAR_1, ..., VAR_n): Gibt die variablen VAR_1, ..., VAR_k wieder frei.

BEFEHL undefall(): Gibt alle variablen wieder frei, denen bisher ein Wert zugewiesen wurde.
