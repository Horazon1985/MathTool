<html><head></head><body><font size="6"><b>Mathematische Operatoren:</b></font><br><br>

MathTool kennt die folgenden Operatoren:<br><br>

<b>OPERATOR diff(FUNCTION, VAR):</b> Berechnet die partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.<br><br>

<b>OPERATOR diff(FUNCTION, VAR, k):</b> Berechnet die k-te partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.<br><br>

<b>OPERATOR diff(FUNCTION, VAR_1, ..., var_k):</b> Berechnet die k-te partielle Ableitung der Funktion FUNCTION nach den Variablen VAR_1, ..., VAR_k. Die Variablen
VAR_i m&uuml;ssen nicht alle verschieden sein.<br><br>

<b>OPERATOR div(FUNCTION):</b> Berechnet die Divergenz der Funktion FUNCTION.<br><br>

<b>OPERATOR gcd(a_1, ..., a_k):</b> Berechnet den gr&ouml;&szlig;ten gemeinsamen Teiler der ganzen Zahlen a_1, ..., a_k.<br><br>

<b>OPERATOR int(FUNCTION, x_1, x_2):</b> Berechnet das bestimmte Integral &uuml;ber die Funktion FUNCTION von x_1 bis x_2.<br><br>

<b>OPERATOR laplace(FUNCTION):</b> Berechnet den Laplace-Operator der Funktion FUNCTION.<br><br>

<b>OPERATOR lcm(a_1, ..., a_k):</b> Berechnet das kleinste gemeinsame Vielfache der ganzen Zahlen a_1, ..., a_k.<br><br>

<b>OPERATOR prod(EXPRESSION, VAR, k_1, k_2):</b> Berechnet das Produkt f&uuml;r den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.<br><br>

<b>OPERATOR sum(EXPRESSION, VAR, k_1, k_2):</b> Berechnet die Summe f&uuml;r den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.<br><br>

<b>OPERATOR taylor(FUNCTION VAR, x_0, k):</b> Berechnet das Taylorpolynom k-ten Grades von der Funktion FUNCTION bez&uuml;glich der Variablen VAR an der Stelle VAR = x_0.<br><br>

<i>Beispiel:</i> taylor(exp(x*y),x,0,5) erzeugt die Ausgabe<br><br> 

1+y*x+(y^2*x^2)/2+(y^3*x^3)/6+(y^4*x^4)/24+(y^5*x^5)/120<br><br>

In der Tat, die Potenzreihenentwicklung von exp(x*y) lautet<br><br>

1+y*x+(y^2*x^2)/2+(y^3*x^3)/6+(y^4*x^4)/24+(y^5*x^5)/120+(y^6*x^6)/720+(y^7*x^7)/5040+...<br><br>

und die Ausgabe liefert hier genau die ersten f&uuml;nf Glieder dieser Reihenetwicklung.<br><br>

Operatoren d&uuml;rfen wiederum Bestandteile weiterer mathematischer Ausdrücke sein und k&ouml;nnen somit weiterverarbeitet werden. So wird beispielsweise der Ausdruck<br><br>

x^2*diff(y^5,y,3) + 6<br><br>

vereinfacht zu<br><br>

60*x^2*y^2+6.
