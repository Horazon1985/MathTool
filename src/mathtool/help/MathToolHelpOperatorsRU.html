Операторы:

MathTool знает такие операторы:

cov([x1, y1], ..., [xn, yn]): Computes the covariance for the values x1, ..., xn and y1, ..., yn.

diff(F(x1, ..., xn), xi): Вычисляет частную производную ∂F/∂xi функции F по переменной xi.

diff(F(x1, ..., xn), xi, k): Вычисляет k-ю частную производную ∂kF/∂xik функции F по переменной xi.

diff(F(x1, ..., xn), xi, ..., xj): Вычисляет частную производную ∂(... ∂F/∂xi)/∂xj функции F по переменнойs xi, ..., xj. Переменные xi, ..., xj не должны быть разными.

div(F(x1, ..., xn)): Вычисляет дивергенцию div(F) = ∂F/∂x1 + ··· + ∂F/∂xn функции F.

fac(n) oder n!: Вычисляет факториал n. Для неотрицательных целых чисел n она возвращает хорошо известный факториал n! = 1·2· ··· ·n числа n. Factorials of negative integers return errors and for factorials of non-integers it returns the value Γ(n + 1) of the Gamma function in approximation mode. Factorials of integers will be computed explicitly only if n <= 10000.

Example: The expression 5! is simplified to

120,


while approx(0.5!) returns

0.8862266958197873


gcd(n1, ..., nk): Вычисляет the greatest common divisor of the integers n1, ..., nk.

grad(F(x1, ..., xn)): Computes the gradient grad(F) = [∂F/∂x1; ··· ; ∂F/∂xn] of the function F.

int(F(x, a1, ..., an), x): Вычисляет the indefinite integral ∫F(x, a1, ..., an)dx функции F(x, a1, ..., an) (по x).

int(F(x, a1, ..., an), x, b1, b2): Вычисляет the definite integral ∫b1b2F(x, a1, ..., an)dx функции F(x, a1, ..., an) from b1 to b2 (по x).

laplace(F(x1, ..., xn)): Вычисляет the Laplace operator ΔF = ∂2F/∂x12 + ··· + ∂2F/∂xn2 функции F.

lcm(n1, ..., nk): Вычисляет the least common multiple of the integers n1, ..., nk.

mod(a, b): If a and b are integers, it returns the remainder r of the division of a by b.

Example: mod(25, 7) returns

4


mu(x1, ..., xn): Вычисляет арефметическое среднее величин x1, ..., xn.

prod(F(k, x1, ..., xn), k, k1, k2): Вычисляет the produkt F(k1, x1, ..., xn)·F(k1 + 1, x1, ..., xn)· ··· ·F(k2, x1, ..., xn). If k1 > k2, then the result is 1.

rot([F1; F2; F3], x, y, z): Computes the curl rot(F) = [∂F/∂y(F3) - ∂F/∂z(F2); -∂F/∂x(F3) + ∂F/∂z(F1) ; ∂F/∂x(F2) - ∂F/∂y(F1)] of the function F.

sigma(x1, ..., xn): Вычисляет стандартное отклонение величин x1, ..., xn.

sum(F(k, x1, ..., xn), k, k1, k2): Вычисляет the sum F(k1, x1, ..., xn) + F(k1 + 1, x1, ..., xn) + ··· + F(k2, x1, ..., xn). If k1 > k2, then the result is 0.

taylor(F(x, y1, ..., yn), x, x0, k): Вычисляет the Taylor polynomial of degree k функции F(x, y1, ..., yn) по переменной x at the point x = x0.

Operators can occur, in contrast to commands, in other mathematical expressions.

Example: The expression

x^2*diff(y^5,y,3) + 6


will be simplified to

60*x^2*y^2+6.

var(x1, ..., xn): Вычисляет дисперсию величин x1, ..., xn.
