Operators:

MathTool knows the following operators:

diff(F(x1, ..., xn), xi): Computes the partial derivative ∂F/∂xi of the function F with respect to the variable xi.

diff(F(x1, ..., xn), xi, k): Computes the k-th partial derivative ∂kF/∂xik of the function F with respect to the variable xi.

diff(F(x1, ..., xn), xi, ..., xj): Computes the partial derivative ∂(... ∂F/∂xi)/∂xj of the function F with respect to the variables xi, ..., xj. The variables xi, ..., xj do not have to be different.

div(F(x1, ..., xn)): Computes the divergence div(F) = ∂F/∂x1 + ··· + ∂F/∂xn of the function F.

fac(n) oder n!: Computes the factorial of n. For nonnegative integers n it returns the well-known factorial n! = 1·2· ··· ·n of n. Factorials of negative integers return errors and for factorials of non-integers it returns the value Γ(n + 1) of the Gamma function in approximation mode. Factorials of integers will be computed explicitly only if n <= 10000.

Example: The expression 5! is simplified to

120,


while approx(0.5!) returns

0.8862266958197873


gcd(n1, ..., nk): Computes the greatest common divisor of the integers n1, ..., nk.

int(F(x, a1, ..., an), x): Computes the indefinite integral ∫F(x, a1, ..., an)dx of the function F(x, a1, ..., an) (with respect to x).

int(F(x, a1, ..., an), x, b1, b2): Computes the definite integral ∫b1b2F(x, a1, ..., an)dx of the function F(x, a1, ..., an) from b1 to b2 (with respect to x).

laplace(F(x1, ..., xn)): Computes the Laplace operator ΔF = ∂2F/∂x12 + ··· + ∂2F/∂xn2 of the function F.

lcm(n1, ..., nk): Computes the least common multiple of the integers n1, ..., nk.

mod(a, b): If a and b are integers, it returns the remainder r of the division of a by b.

Example: mod(25, 7) returns

4


prod(F(k, x1, ..., xn), k, k1, k2): Computes the produkt F(k1, x1, ..., xn)·F(k1 + 1, x1, ..., xn)· ··· ·F(k2, x1, ..., xn). If k1 > k2, then the result is 1.

sum(F(k, x1, ..., xn), k, k1, k2): Computes the sum F(k1, x1, ..., xn) + F(k1 + 1, x1, ..., xn) + ··· + F(k2, x1, ..., xn). If k1 > k2, then the result is 0.

taylor(F(x, y1, ..., yn), x, x0, k): Computes the Taylor polynomial of degree k of the function F(x, y1, ..., yn) with respect to the variable x at the point x = x0.

Operators can occur, in contrast to commands, in other mathematical expressions.

Example: The expression

x^2*diff(y^5,y,3) + 6


will be simplified to

60*x^2*y^2+6.
