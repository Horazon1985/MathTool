<html><head></head><body><font size="6"><b>Mathematische Operatoren:</b></font><br><br>

MathTool kennt die folgenden Operatoren:<br><br>

<b>diff(FUNCTION, VAR):</b> Berechnet die partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.<br><br>

<b>diff(FUNCTION, VAR, k):</b> Berechnet die k-te partielle Ableitung der Funktion FUNCTION nach der Variablen VAR.<br><br>

<b>diff(FUNCTION, VAR_1, ..., var_k):</b> Berechnet die k-te partielle Ableitung der Funktion FUNCTION nach den Variablen VAR_1, ..., VAR_k. Die Variablen
VAR_i m&uuml;ssen nicht alle verschieden sein.<br><br>

<b> div(FUNCTION):</b> Berechnet die Divergenz der Funktion FUNCTION.<br><br>

<b>fac(n) oder n!:</b> Berechnet die Fakult&auml;t der Zahl n. Fakult&auml;ten negativer ganzer Zahlen liefern einen Fehler. Fakult&auml;ten von nicht-ganzen Zahlen werden im Approximationsmodus
durch den Wert der <i>Gammafunktion</i> Gamma(n + 1) approximiert. Fakult&auml;ten ganzer Zahlen werden nur bis n <= 10000 ausgeschrieben.<br><br>

<i>Beispiel:</i> Der Ausdruck 5! erzeugt die Ausgabe<br><br> 

120,<br><br>

w&auml;hrend der Ausdruck approx(0.5!) zu<br><br>

0.8862266958197873<br><br>

vereinfacht wird.<br><br>

<b>gcd(n_1, ..., n_k):</b> Berechnet den gr&ouml;&szlig;ten gemeinsamen Teiler der ganzen Zahlen n_1, ..., n_k.<br><br>

<b>int(FUNCTION, x_1, x_2):</b> Berechnet das bestimmte Integral &uuml;ber die Funktion FUNCTION von x_1 bis x_2.<br><br>

<b>laplace(FUNCTION):</b> Berechnet den Laplace-Operator der Funktion FUNCTION.<br><br>

<b>lcm(n_1, ..., n_k):</b> Berechnet das kleinste gemeinsame Vielfache der ganzen Zahlen n_1, ..., n_k.<br><br>

<b>mod(a, b):</b> Sind a und b ganze Zahlen, so berechnet es den Divisionsrest r bei ganzzahliger Division von a durch b.<br><br>

<i>Beispiel:</i> mod(25, 7) erzeugt die Ausgabe<br><br>

4.<br><br>

<b>prod(EXPRESSION, VAR, k_1, k_2):</b> Berechnet das Produkt f&uuml;r den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.<br><br>

<b>sum(EXPRESSION, VAR, k_1, k_2):</b> Berechnet die Summe f&uuml;r den Ausdruck EXPRESSION von VAR = k_1 bis VAR = k_2.<br><br>

<b>taylor(FUNCTION VAR, x_0, k):</b> Berechnet das Taylorpolynom k-ten Grades von der Funktion FUNCTION bez&uuml;glich der Variablen VAR an der Stelle VAR = x_0.<br><br>

Operatoren k&ouml;nnen wiederum in weiteren mathematischen Ausdr&uuml;cken vorkommen und somit weiterverarbeitet werden. 

<i>Beispiel:</i> Der Ausdruck<br><br>

x^2*diff(y^5,y,3) + 6<br><br>

erzeugt die Ausgabe<br><br>

60*x^2*y^2+6.
