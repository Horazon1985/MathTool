Commands:

MathTool knows the following console commands:

approx(F(x1, ..., xn)): Approximates the expression F(x1, ..., xn).

Example: approx(3^0.5) returns

1.7320508075688772


clear(): Clears the entire console. However, variables and functions defined by user stay in memory.

def(x = x0): Assigns to variable x the constant value x0.

def(F(x1, ..., xn) = G(x1, ..., xn)): Defines a function F with variables x1, ..., xn by the expression G(x1, ..., xn). Of course, predefined functions like sin(), tan(), exp(), etc. cannot be overriden.

Example: After the input def(f(x, y) = x^2 + sin(x*y)), MathTool knows the function f. The input f(u^3, u*v^4) then returns

u^6+sin(u^4*v^4).


defvars(): Returns the list of all variables with predefined values using command def().

euler(n): Returns the first n digits of Eulers constant e. The result is not roundet.

Example: euler(20) returns

2.71828182845904523536


For memory reasons the number n is not allowed to exceed 100000.

expand(F(x1, ..., xn)): Expands the expression F.

Example: expand(x+(a+b)*(c+d+e)) returns

x+a*c+a*d+a*e+b*c+b*d+b*e


latex(F1(x1, ..., xn) = ... = Fm(x1, ..., xn)): Generates the Latex code for the mathematical expression F1(x1, ..., xn) = ... = Fm(x1, ..., xn).

pi(n): Returns the first n digits of the constant pi. The result is not roundet.

Example: pi(20) returns

3.14159265358979323846


For memory reasons the number n is not allowed to exceed 100000.

plot2d(F1(x), ..., Fn(x), x1, x2): Plots the graphs of the functions F1, ..., Fn for x1 <= x <= x2. The zoom factor can be (almost) arbitrary.

plot2d(F1(x, y) = F2(x, y), x1, x2, y1, y2): Plots the solution of the equation F1(x, y) = F2(x, y) in two variables for x1 <= x <= x2, y1 <= y <= y2. Here it is not possible to zoom in or to zoom out as well as to move the coordinate system.
Wichtig: Die Grenzen x1, x2, y1, y2 sollen so angegeben werden, dass x, y die alphabetische Reihenfolge der Variablen ist.

plot3d(F(x, y), x1, x2, y1, y2): Zeichnet den Graphen der Funktion F(x, y) im Rechteck x1 <= x <= x2, y1 <= y <= y2.
Important: The limits x1, x2, y1, y2 have to be chosen in a way such that x, y is the alphabetical order of the variables. The zoom factor lies between 0.1 and 10.

plotcurve((F1(t), F2(t)), t1, t2): Zeichnet die ebene Kurve, welche durch die Komponentenfunktionen F1 und F2 parametrisiert wird, für t1 <= t <= t2. Der Zoomfaktor liegt hier zwischen 0.1 und 10. Manchmal kann es vorkommen (etwa beim Zeichnen von Kurven, welche sehr "unglücklich" parametrisiert wurden), dass nicht die ganze Kurve, sondern nur ein Teil von ihr gezeichnet werden (etwa für t1 <= t <= t' für ein t' <= t2).

plotcurve((F1(t), F2(t), F3(t)), t1, t2): Zeichnet die räumliche Kurve, welche durch die Komponentenfunktionen F1, F2 und F3 parametrisiert wird, für t1 <= t <= t2. Der Zoomfaktor liegt hier zwischen 0.1 und 10. Auch hier gilt dasselbe wie im Falle von plotcurve im Zweidimensionalen, dass manche "schlecht" parametrisierten Kurven nicht komplett gezeichnet werden können.

plotpolar(F1(t), ..., Fn(t), t1, t2): Zeichnet den Graphen der Funktionen F1, ..., Fn für t1 <= t <= t2, wobei t den Winkel und Fi(t) den Radius in Polarkoordinaten angeben. Der Zoomfaktor liegt hier zwischen 0.1 und 10.

solve(F1(x) = F2(x), x1, x2): Bestimmt die numerischen Lösungen der Gleichung F1(x) = F2(x) im Bereich x1 <= x <= x2.

solve(F1(x) = F2(x), x1, x2, n): Bestimmt die numerischen Lösungen der Gleichung F1(x) = F2(x) im Bereich x1 <= x <= x2. Hierbei wird das Intervall [x1, x2] in n gleiche Teile unterteilt und in jedem Teilintervall wird höchstens eine Lösung ermittelt. Höheres n kann zwar potentiell mehr Lösungen liefern, verlängert dafür aber die Berechnungszeit. Diese Funktion kann dann nützlich sein, wenn der Benutzen über bestimmte Kenntnissebezüglich der Lösungen der Gleichung F1(x) = F2(x) verfügt.

solvedeq(F(x, y, y', y'', ..., yn-1), x, n, x0, x1, y0, y1, ..., yn-1): Liefert eine numerische Lösung der Diferentialgleichung yn = F(x, y, y', y'', ..., yn-1), y(x0) = y0, ..., yn-1(x0) = yn-1 im Bereich x_0 <= x <= x1. Für die numerische Berechnung wird das Intervall [x0, x1] in 1000 gleiche Teile unterteilt und in der Konsole werden alle 1000 Funktionswerte an den betreffenden Stellen ausgegeben.

Example: solvedeq(-y,t,2,0,10,1,0) liefert eine numerische Lösung der Differentialgleichung y''(t) = -y(t), y(0) = 1, y'(0) = 0 im Bereich 0 <= t <= 10. Hierbei werden 1000 Funktionswerte von y(t) ausgegeben; die Schrittweite beträgt daher 0.01.

tangent(F(x1, ..., xn), x1 = a1, ..., xn = an): Liefert die Gleichung der Tangentialebene an den Graphen von F im Punkte (x1, ..., xn) = (a1, ..., an).

Example: tangent(x^3+x/2+3,x=2) liefert die folgende Gleichung:

Y=12+(25*(x-2))/2.


Zusätzlich werden beide Graphen (der von x^3+x/2+3 und von 12+25*(x-2)/2) gezeichnet.

taylordeq(F(x, y, y', y'', ..., yn-1), x, n, x0, y0, y1, ..., yn-1, m): Returns the Taylor polynomial of degree m of the solution of the differential equation yn = F(x, y, y', y'', ..., yk), y(x0) = y_0, ..., yn-1(x0) = yn-1.

Example: taylordeq(-y,t,2,0,1,0,10) returns the Taylor polynomial of degree 10 of the solution of the differential equation y''(t) = -y(t), y(0) = 1, y'(0) = 0. The exact solution is given by y(t) = cos(t), hence the power series is

y(t) = cos(t) = 1 - t^2/2! + t^4/4! - t^6/6! + ...


Thus, MathTool returns

1 + t^4/24 + t^8/40320-(t^2/2 + t^6/720 + t^10/3628800).


undef(x1, ..., xn): The variables x1, ..., xn are again indeterminates, that is, without value assigned.

undefall(): All variables, which are assigned a value during this session, are again indeterminates.

All console commands that were entered during a session are stored and can be retrieved using the arrow keys (UP = last command, DOWN = next command).
