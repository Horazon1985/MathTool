Commands:

MathTool knows the following console commands:

approx(F(x1, ..., xn)): Approximates the expression F(x1, ..., xn).

Example: approx(3^0.5) returns

1.7320508075688772


ccnf(F(x1, ..., xn)): Returns the canonical conjunctive normal form of the logical expression F(x1, ..., xn).

cdnf(F(x1, ..., xn)): Returns the canonical disjunctive normal form of the logical expression F(x1, ..., xn).

clear(): Clears the entire console. However, variables and functions defined by user stay in memory.

def(x = x0): Assigns to variable x the constant value x0.

def(F(x1, ..., xn) = G(x1, ..., xn)): Defines a function F with variables x1, ..., xn by the expression G(x1, ..., xn). Of course, predefined functions like sin(), tan(), exp(), etc. cannot be overriden.

Example: After the input def(f(x, y) = x^2 + sin(x*y)), MathTool knows the function f. The input f(u^3, u*v^4) then returns

u^6+sin(u^4*v^4).


defvars(): Returns the list of all variables with predefined values using command def().

eigenvalues(A): Returns the eigenvalues of the quadratic matrix A, if they are computable.

eigenvectors(A): Returns the eigenvectors of the quadratic matrix A, grouped by the corresponding eigenvalues of A, if they are computable.

euler(n): Returns the first n digits of Eulers constant e. The result is not roundet.

Example: euler(20) returns

2.71828182845904523536


For memory reasons the number n is not allowed to exceed 100000.

expand(F(x1, ..., xn)): Expands the expression F.

Example: expand(x+(a+b)*(c+d+e)) returns

x+a*c+a*d+a*e+b*c+b*d+b*e


latex(F1(x1, ..., xn) = ... = Fm(x1, ..., xn)): Generates the Latex code for the mathematical expression F1(x1, ..., xn) = ... = Fm(x1, ..., xn).

pi(n): Returns the first n digits of the constant pi. The result is not roundet.

Example: pi(20) returns

3.14159265358979323846


For memory reasons the number n is not allowed to exceed 100000.

plot2d(F1(x), ..., Fn(x), x, x1, x2): Plots the graphs of the functions F1, ..., Fn for x1 <= x <= x2. The zoom factor can be (almost) arbitrary.

plotimplicit2d(F1(x, y) = F2(x, y), x, y, x1, x2, y1, y2): Plots the solution of the equation F1(x, y) = F2(x, y) in two variables for x1 <= x <= x2, y1 <= y <= y2. Here it is not possible to zoom in or to zoom out as well as to move the coordinate system.
plot3d(F1(x, y), ..., Fn(x, y), x, y, x1, x2, y1, y2): Plots the graphs of the functions F1, ..., Fn in the rectangle x1 <= x <= x2, y1 <= y <= y2. The zoom factor can attain all values between 0.1 and 10.

plotimplicit3d(F1(x, y, z) = F2(x, y, z), x, y, z, x1, x2, y1, y2, z1, z2): Plots the solution of the equation F1(x, y, z) = F2(x, y, z) in three variables in the cuboid x1 <= x <= x2, y1 <= y <= y2, z1 <= z <= z2. The zoom factor can attain all values between 0.1 and 10.
plotcurve2d([F1(t); F2(t)], t, t1, t2): Plots the plane curve which is given by the component functions x = F1(t) and y = F2(t) for t1 <= t <= t2. The zoom factor can attain all values between 0.1 and 10. Sometimes it may happen (if choosing "bad" curves) that only a part of the is drawn (that is, for t1 <= t <= t' for some t' <= t2).

plotcurve3d([F1(t); F2(t); F3(t)], t, t1, t2): Plots the curve which is given by the component functions x = F1(t), y = F2(t) and z = F3(t) for t1 <= t <= t2. The zoom factor can attain all values between 0.1 and 10. Sometimes it may happen (if choosing "bad" curves), just as in the case of plane curves, that only a part of the is drawn.

plotpolar(F1(t), ..., Fn(t), t, t1, t2): Plots the graph of the functions F1, ..., Fn for t1 <= t <= t2, where t is the angle and Fi(t) is the radius in polar coordinates. The zoom factor can attain all values between 0.1 and 10.

plotcylindrical(F1(r, t), ..., Fn(r, t), r, t, r1, r2, t1, t2): Plots the graphs of the functions F1(r, t), ..., Fn(r, t) for r1 <= r <= r2 and t1 <= t <= t2 in cylindrical coordinates, where r is the radius und t is the angle. The radius must be non-negative the difference t2 - t1 of the angles must be at most 20π because of technical reasons. The zoom factor can attain all values between 0.1 and 10.

plotspherical(F1(u, v), ..., Fn(u, v), u, v, u1, u2, v1, v2): Plots the graphs of the functions F1(u, v), ..., Fn(u, v) for u1 <= u <= u2 and v1 <= v <= v2 in spherical coordinates, where u is the angle in the x, y-plane and v is the angle to the z-axis. The differences u2 - u1 and v2 - v1 of the angles must be at most 20π because of technical reasons. The zoom factor can attain all values between 0.1 and 10.

plotsurface([F1(s, t); F2(s, t); F3(s, t)], s, t, s1, s2, t1, t2): Plots the surface which is given by the component functions x = F1(s, t), y = F2(s, t) and z = F3(s, t) for s1 <= s <= s2 and t1 <= t <= t2. The zoom factor can attain all values between 0.1 and 10.

plotvectorfield2d([F1(x, y); F2(x, y)], x, y, x1, x2, y1, y2): Plots the vector field [F1(x, y); F2(x, y)] for x1 <= x <= x2 and y1 <= y <= y2. The zoom factor can be (almost) arbitrary.

regressionline(P1, ..., Pn): Computes the equation of the regression line for the points P1, ..., Pn, 2 <= n, and also plots it, if possible. The points Pi must be (after simplifying) of the form [a;b], i.e. they must be (2x1)-matrices.

solve(F1(x) = F2(x)): Returns the algebraic (exact) solutions of the equation F1(x) = F2(x) in x, if possible. Note, that not every equation can be solved exactly. For example, equations of the form exp(x) - x = 3 do not admit solutions which can be expressed in algebraic terms or elementary functionsl. In these cases, MathTool will not return any (exact) solution. Example: Consider the equation x^18 - 3*x^6 - 11 = 0. The command solve(x^18 - 3*x^6 - 11 = 0) returns the solutions

x_1 = ((11/2+117^(1/2)/2)^(1/3)+(11/2-117^(1/2)/2)^(1/3))^(1/6)

x_2 = -((11/2+117^(1/2)/2)^(1/3)+(11/2-117^(1/2)/2)^(1/3))^(1/6).

If approximated, these solutions are x1, 2 ≈ ±1.1777403511345441. This is also returned by the solve command with suitable interval (see below).

solve(F1(x, a1, ..., an) = F2(x, a1, ..., an), x): Returns the (exact) solutions of the equation F1(x, a1, ..., an) = F2(x, a1, ..., an) in x (depending on the parameters a1, ..., an), if possible.

solve(F1(x) = F2(x), x1, x2): Returns the numerical solutions of the equation F1(x) = F2(x) in the interval x1 <= x <= x2.

solve(F1(x) = F2(x), x1, x2, n): Returns the numerical solutions of the equation F1(x) = F2(x) in the interval x1 <= x <= x2. Here, the interval [x1, x2] is subdivided in n equal subintervals and in every subinterval at most one solution can be found. Higher n may potentially yield more solutions, but it increases the computation time. This function can be useful if the user has specific knowlegde of the solutions of the equation F1(x) = F2(x).

solvediffeq(F1(x, y, y', y'', ..., y(n)) = F2(x, y, y', y'', ..., y(n)), x, y): Returns the solution of the differential equation F1(x, y, y', y'', ..., y(n)) = F2(x, y, y', y'', ..., y(n)).

solvediffeq(F(x, y, y', y'', ..., y(n-1)), x, y, n, x0, x1, y0, y1, ..., yn-1): Returns the numerical solution of the differential equation y(n) = F(x, y, y', y'', ..., y(n-1)), y(x0) = y0, ..., y(n-1)(x0) = yn-1 in the interval x_0 <= x <= x1. For the numerical computation, the interval [x0, x1] is subdivided in 1000 subintervals and MathTool returns all 1001 function values in the corresponding points.

Example: solvedeq(-y,t,y,2,0,10,1,0) returns the numerical solutions of the differential equation y''(t) = -y(t), y(0) = 1, y'(0) = 0 for 0 <= t <= 10. Here, MathTool returns 1001 values of y(t), hence the increment is 0.01.

tangent(F(x1, ..., xn), x1 = a1, ..., xn = an): Returns the equation of the tangent space to the graph of F at the point (x1, ..., xn) = (a1, ..., an).

Example: tangent(x^3+x/2+3,x=2) returns the following equation:

Y=12+(25*(x-2))/2.


In addition, MathTool plots both graphs (those of x^3+x/2+3 and of 12+25*(x-2)/2).

taylordiffeq(F(x, y, y', y'', ..., y(n-1)), x, y, n, x0, y0, y1, ..., yn-1, m): Returns the Taylor polynomial of degree m of the solution of the differential equation y(n) = F(x, y, y', y'', ..., y(n-1)), y(x0) = y_0, ..., y(n-1)(x0) = yn-1.

Example: taylordeq(-y,t,y,2,0,1,0,10) returns the Taylor polynomial of degree 10 of the solution of the differential equation y''(t) = -y(t), y(0) = 1, y'(0) = 0. The exact solution is given by y(t) = cos(t), hence the power series is

y(t) = cos(t) = 1 - t^2/2! + t^4/4! - t^6/6! + ...


Thus, MathTool returns

1 + t^4/24 + t^8/40320-(t^2/2 + t^6/720 + t^10/3628800).


undeffuncs(f1, ..., fn): Removes the functions f1, ..., fn.

undefvars(x1, ..., xn): The variables x1, ..., xn are again indeterminates, that is, without value assigned.

undefallfuncs(): Removes all functions, which were defined during this session.

undefallvars(): All variables, which are assigned a value during this session, are again indeterminates.

undefall(): Removes all functions and all variables, which were defined during this session.

All console commands that were entered during a session are stored and can be retrieved using the arrow keys (UP = last command, DOWN = next command).
