Оператори:

MathTool знає такі оператори:

cov([x1, y1], ..., [xn, yn]): Обчислює коваріацію значень x1, ..., xn та y1, ..., yn.

diff(F(x1, ..., xn), xi): Обчислює часткову похідну ∂F/∂xi функції F по змінній xi.

diff(F(x1, ..., xn), xi, k): Обчислює k-ту часткову похідну ∂kF/∂xik функції F по змінній xi.

diff(F(x1, ..., xn), xi, ..., xj): Обчислює часткову похідну ∂(... ∂F/∂xi)/∂xj функції F по зміннійs xi, ..., xj. Змінні xi, ..., xj не мають бути різними.

div(F(x1, ..., xn), x1, ..., xn): Обчислює дивергенцію div(F) = ∂F/∂x1 + ··· + ∂F/∂xn функції F.

fac(n) oder n!: Обчислює факторіал n. Для невід'ємного n вона повертає добре відомий факторіал n! = 1·2· ··· ·n від n. Факторіали від від'ємних чисел повертають помилки, а факторіла від нецілих чисел, повертають значення Γ(n + 1) Гама функції у наближеному вигляді. Факторіали цілих чисел будуть обчислені точно тільки для n <= 10000.

Приклад: Вираз 5! буде спрощено до

120,


в той час як approx(0.5!) повертає

0.8862266958197873


gcd(n1, ..., nk): Обчислює найбільший спільний дільник цілих чисел n1, ..., nk.

grad(F(x1, ..., xn)): Обчислює градієнт grad(F) = [∂F/∂x1; ··· ; ∂F/∂xn] функції F.

int(F(x, a1, ..., an), x): Обчислює невизначений інтеграл ∫F(x, a1, ..., an)dx функції F(x, a1, ..., an) (по x).

int(F(x, a1, ..., an), x, b1, b2): Обчислює визначений інтеграл ∫b1b2F(x, a1, ..., an)dx функції F(x, a1, ..., an) від b1 до b2 (по x).

laplace(F(x1, ..., xn), x1, ..., xn): Обчислює оператор Лапласа ΔF = ∂2F/∂x12 + ··· + ∂2F/∂xn2 функції F.

lcm(n1, ..., nk): Обчислює найменше спільне кратне цілих чисел n1, ..., nk.

max(x1, ..., xn): Обчислює максимум величин x1, ..., xn.

min(x1, ..., xn): Обчислює мінімум величин x1, ..., xn.

mod(a, b): Якщо a та b - цілі числа, то вона повертає остачу r від ділення a на b.

Приклад: mod(25, 7) повертає

4


mu(x1, ..., xn): Обчислює середнє арефметичне величин x1, ..., xn.

prod(F(k, x1, ..., xn), k, k1, k2): Обчислює добуток F(k1, x1, ..., xn)·F(k1 + 1, x1, ..., xn)· ··· ·F(k2, x1, ..., xn). Якщо k1 > k2, тоді результат дорівнює 1.

rot([F1; F2; F3], x, y, z): Обчислює ротор rot(F) = [∂F/∂y(F3) - ∂F/∂z(F2); -∂F/∂x(F3) + ∂F/∂z(F1) ; ∂F/∂x(F2) - ∂F/∂y(F1)] функції F.

sigma(x1, ..., xn): Обчислює стандартне відхилення величин x1, ..., xn.

sum(F(k, x1, ..., xn), k, k1, k2): Обчислює суму F(k1, x1, ..., xn) + F(k1 + 1, x1, ..., xn) + ··· + F(k2, x1, ..., xn). If k1 > k2, тоді результат дорівнює 0.

taylor(F(x, y1, ..., yn), x, x0, k): Обчислює поліном Тейлора степеня k функції F(x, y1, ..., yn) по змінній x у точці x = x0.

Оператор, на відміну від команди, може використовуватися у інших математичних виразах.

Приклад: Вираз

x^2*diff(y^5,y,3) + 6


буде спрощено до

60*x^2*y^2+6.

var(x1, ..., xn): Обчислює дисперсію величин x1, ..., xn.
