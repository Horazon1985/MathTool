Операторы:

MathTool знает такие операторы:

cov([x1, y1], ..., [xn, yn]): Вычисляет ковариацию величин x1, ..., xn и y1, ..., yn.

diff(F(x1, ..., xn), xi): Вычисляет частную производную ∂F/∂xi функции F по переменной xi.

diff(F(x1, ..., xn), xi, k): Вычисляет k-ю частную производную ∂kF/∂xik функции F по переменной xi.

diff(F(x1, ..., xn), xi, ..., xj): Вычисляет частную производную ∂(... ∂F/∂xi)/∂xj функции F по переменнойs xi, ..., xj. Переменные xi, ..., xj не должны быть разными.

div(F(x1, ..., xn), x1, ..., xn): Вычисляет дивергенцию div(F) = ∂F/∂x1 + ··· + ∂F/∂xn функции F.

fac(n) oder n!: Вычисляет факториал n. Для неотрицательных целых чисел n она возвращает хорошо известный факториал n! = 1·2· ··· ·n числа n. Факториалы отрицательных чисел возвращают ошибки, а факториал нецелых чисел возвращает значение Γ(n + 1) Гамма функции в режиме аппроксимации. Факториал целого числа вычисляется в явном виде, только если n <= 10000.

Пример: Выражение 5! упрощается к

120,


поскольку approx(0.5!) возвращает

0.8862266958197873


gcd(n1, ..., nk): Вычисляет наибольший общий делитель целых чисел n1, ..., nk.

grad(F(x1, ..., xn)): Вычисляет градиент grad(F) = [∂F/∂x1; ··· ; ∂F/∂xn] функции F.

int(F(x, a1, ..., an), x): Вычисляет неопределённый интеграл ∫F(x, a1, ..., an)dx функции F(x, a1, ..., an) (по x).

int(F(x, a1, ..., an), x, b1, b2): Вычисиляет определённый интеграл ∫b1b2F(x, a1, ..., an)dx функции F(x, a1, ..., an) от b1 до b2 (по x).

laplace(F(x1, ..., xn), x1, ..., xn): Вычисляет ооператор Лапласа ΔF = ∂2F/∂x12 + ··· + ∂2F/∂xn2 функции F.

lcm(n1, ..., nk): Вычисляет наименьшее общее кратное целых чисел n1, ..., nk.

max(x1, ..., xn): Вычисляет максимум величин x1, ..., xn.

min(x1, ..., xn): Вычисляет минимум величин x1, ..., xn.

mod(a, b): Если a и b целые числа, он возвращает остачу r от деления a на b.

Пример: mod(25, 7) возвращает

4


mu(x1, ..., xn): Вычисляет арефметическое среднее величин x1, ..., xn.

prod(F(k, x1, ..., xn), k, k1, k2): Вычисляет произведение F(k1, x1, ..., xn)·F(k1 + 1, x1, ..., xn)· ··· ·F(k2, x1, ..., xn). Если k1 > k2, тогда результат равен 1.

rot([F1; F2; F3], x, y, z): Вычисляет ротор rot(F) = [∂F/∂y(F3) - ∂F/∂z(F2); -∂F/∂x(F3) + ∂F/∂z(F1) ; ∂F/∂x(F2) - ∂F/∂y(F1)] функции F.

sigma(x1, ..., xn): Вычисляет стандартное отклонение величин x1, ..., xn.

sum(F(k, x1, ..., xn), k, k1, k2): Вычисляет суммуу F(k1, x1, ..., xn) + F(k1 + 1, x1, ..., xn) + ··· + F(k2, x1, ..., xn). Если k1 > k2, тогда 0.

taylor(F(x, y1, ..., yn), x, x0, k): Вычисляет полином Тейлора степеня k функции F(x, y1, ..., yn) по переменной x at the point x = x0.

Операторы, в отличие от команд, могут встречаться в математических выражениях.

Пример: Выражение

x^2*diff(y^5,y,3) + 6


будет упрощено до

60*x^2*y^2+6.

var(x1, ..., xn): Вычисляет дисперсию величин x1, ..., xn.
