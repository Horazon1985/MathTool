Команди:

MathTool знає такі консольні команди:

approx(F(x1, ..., xn)): Наближує вираз F(x1, ..., xn).

Приклад: approx(3^0.5) повертає

1.7320508075688772


ccnf(F(x1, ..., xn)): Повертає канонічну кон'юнктивну нормальну форму логічного выразу F(x1, ..., xn).

cdnf(F(x1, ..., xn)): Повертає канонічну диз'юнктивну нормальну форму логічного выразу F(x1, ..., xn).

clear(): Очищує всю консоль. Причому змінні та функції визначені користувачем лишаються у пам'яті.

def(x = x0): Присвоює змінній x константу x0.

def(F(x1, ..., xn) = G(x1, ..., xn)): Визначає функцію F від змінних x1, ..., xn за допомогою виразу G(x1, ..., xn). Звичайно наперед задані функції, такі як sin(), tan(), exp(), і.т.д., не можуть бути перевизначені.

Приклад: Після виконання def(f(x, y) = x^2 + sin(x*y)), MathTool знає функцію f. Тоді f(u^3, u*v^4) повертатиме

u^6+sin(u^4*v^4).


defvars(): Повертає список усіх змінних та їх значень, що були визначені командою def().

eigenvalues(A): Повертає власні числа квадратної матриці A, якщо їх можливо обчислити.

eigenvectors(A): Повертає власні вектори квадратної матриці A, погруповані по відповідним власним числам, якщо їх можливо обчислити.

euler(n): Повертає перші n розрядів константи e. Результат не округляється.

Приклад: euler(20) повертає

2.71828182845904523536


З міркувань оптимального використання пам'ті n не повинно перевищувати 100000.

expand(F(x1, ..., xn)): Розкриває всі дужки у виразі F.

Приклад: expand(x+(a+b)*(c+d+e)) повертає

x+a*c+a*d+a*e+b*c+b*d+b*e


latex(F1(x1, ..., xn) = ... = Fm(x1, ..., xn)): Генерує Latex код для математичного виразу F1(x1, ..., xn) = ... = Fm(x1, ..., xn).

normal(F(x1, ..., xn), x1 = a1, ..., xn = an): Повертає параметризацію нормалі до графа F у точці (x1, ..., xn) = (a1, ..., an).

pi(n): Повертає перші n розрядів константи π. Результат не округляється.

Приклад: pi(20) повертає

3.14159265358979323846


З міркувань оптимального використання пам'ті n не повинно перевищувати 100000.

plot2d(F1(x), ..., Fn(x), x, x1, x2): Будує графіки функцій F1, ..., Fn від змінних x1 <= x <= x2. Коефіцієнт масштабування може бути (майже) довільним.

plotimplicit2d(F1(x, y) = F2(x, y), x, y, x1, x2, y1, y2): Будує графік розв'язку рівняння F1(x, y) = F2(x, y) двох змінних x1 <= x <= x2, y1 <= y <= y2. Тут масштабування та рух системи координат неможливі.
plot3d(F1(x, y), ..., Fn(x, y), x, y, x1, x2, y1, y2): Будує графік функцій F1, ..., Fn в прямокутнику x1 <= x <= x2, y1 <= y <= y2. Коефіцієнт масштабуванняcan може приймати всі значення між 0.1 та 10.

plotimplicit3d(F1(x, y, z) = F2(x, y, z), x, y, z, x1, x2, y1, y2, z1, z2): Будує графік розв'язку рівняння F1(x, y, z) = F2(x, y, z) у трьох змінних у кубоїді x1 <= x <= x2, y1 <= y <= y2, z1 <= z <= z2. Коефіцієнт масштабуванняcan може приймати всі значення між 0.1 та 10.
plotcurve2d([F1(t); F2(t)], t, t1, t2): Будує плоску криву, що параметризована функціями x = F1(t) та y = F2(t) від t1 <= t <= t2. Коефіцієнт масштабуванняcan може приймати всі значення між 0.1 та 10. Може так статися (якщо вибрати "погану" криву), що буде зображено лише частину кривої (тобто, для t1 <= t <= t' для деякого t' <= t2).

plotcurve3d([F1(t); F2(t); F3(t)], t, t1, t2): Будує криву, що задана функціями x = F1(t), y = F2(t) та z = F3(t) від t1 <= t <= t2. Коефіцієнт масштабуванняcan може приймати всі значення між 0.1 та 10. Може так статися (якщо вибрати "погану" криву), так само як у випадку плоскої кривої, що буде зображено лише частину кривої.

plotpolar(F1(t), ..., Fn(t), t, t1, t2): Будує графіки функцій F1, ..., Fn від t1 <= t <= t2, де t - це кут та Fi(t) - це радіус у полярних координатах. Коефіцієнт масштабуванняcan може приймати всі значення між 0.1 та 10.

plotcylindrical(F1(r, t), ..., Fn(r, t), r, t, r1, r2, t1, t2): Будує графіки функцій F1(r, t), ..., Fn(r, t) для r1 <= r <= r2 та t1 <= t <= t2 у циліндричних координатах, де r це радіус, а t - кут. Радіус має бути невід'ємним, різниця t2 - t1 кутів не має перевищувати 20π з технічних причин. Фактор маштабування може набувати усіх значень між 0.1 та 10.

plotspherical(F1(u, v), ..., Fn(u, v), u, v, u1, u2, v1, v2):Будує графіки функцій F1(u, v), ..., Fn(u, v) для u1 <= u <= u2 та v1 <= v <= v2 у циліндричних координатах, де u це кут у x, y-площині та v - кут до осі z. Різниці u2 - u1 та v2 - v1 кутів не мають перевищувати 20π з технічних причин. Фактор маштабування може набувати усіх значень між 0.1 та 10.

plotsurface([F1(s, t); F2(s, t); F3(s, t)], s, t, s1, s2, t1, t2): Будує поверхню, що задана компонентними функціями x = F1(s, t), y = F2(s, t) and z = F3(s, t) для s1 <= s <= s2 та t1 <= t <= t2. Коефіцієнт масштабуванняcan може приймати всі значення між 0.1 та 10.

plotvectorfield2d([F1(x, y); F2(x, y)], x, y, x1, x2, y1, y2): Будує векторне поле [F1(x, y); F2(x, y)] для x1 <= x <= x2 та y1 <= y <= y2. Фактор маштабування може бути (майже) будь яким.

plotvectorfieldpolar([F1(r, t); F2(r, t)], r, t, r1, r2, t1, t2): Будує векторне поле [F1(r, t); F2(r, t)] для r1 <= r <= r2 та t1 <= t <= t2. Тут, (r, t) це полярне зображення поточної точки та F1(r, t) це радіус та F2(r, t) це кут полярного зображення векторного поля. Фактор маштабування може бути (майже) будь яким.

regressionline(P1, ..., Pn): Обчислює рівняння регресійної прямої для точок P1, ..., Pn, 2 <= n, а також будує графік, якщо це можливо. Точки Pi мають бути (післіа спрощення) у формі [a;b], тобто мають бути (2x1)-матрицями.

solve(F1(x) = F2(x)): Повертає алгебраїчні (точні) розв'язки рівняння F1(x) = F2(x) у x, якщо це можливо. Помітимо, що не кожне рівняння може бути розв'язано точно. Наприклад, рівняння виду exp(x) - x = 3 не має розв'язків, що що можна виразити алгебраїчно або від елементарних функцій. В цьому випадку MathTool не поверне (точних) розв'язків. Приклад: Розглянемо рівняння x^18 - 3*x^6 - 11 = 0. Команда solve(x^18 - 3*x^6 - 11 = 0) повертає розв'язки

x_1 = ((11/2+117^(1/2)/2)^(1/3)+(11/2-117^(1/2)/2)^(1/3))^(1/6)

x_2 = -((11/2+117^(1/2)/2)^(1/3)+(11/2-117^(1/2)/2)^(1/3))^(1/6).

У наближеному вигляді, ці розв'язки є x1, 2 ≈ ±1.1777403511345441. Те саме поверне команда з відповідним інтервалом (дивіться ниже).

solve(F1(x, a1, ..., an) = F2(x, a1, ..., an), x): Повертає (точні) розв'язки рівняння F1(x, a1, ..., an) = F2(x, a1, ..., an) від x (в залежності від параметрів a1, ..., an), якщо це можливо.

solve(F1(x) = F2(x), x1, x2): Повертає числові розв'язки рівняння F1(x) = F2(x) у інтервалі x1 <= x <= x2.

solve(F1(x) = F2(x), x1, x2, n): Повертає числові розв'язки рівняння F1(x) = F2(x) у інтервалі x1 <= x <= x2. Тут інтервал [x1, x2] розбивається на n рівних підінтервалів, і у кожному підінтервалі може бути знайдено максимум один розв'язок. Збільшення n може потенційно породити більше розв'язків, але це збільшує час підрахунку. Ця функція може бути корисною, якщо користувач має конкретні знання про розв'язки рівняння F1(x) = F2(x).

solvediffeq(F1(x, y, y', y'', ..., y(n)) = F2(x, y, y', y'', ..., y(n)), x, y): Повертає розв'язки диференціального рівняння F1(x, y, y', y'', ..., y(n)) = F2(x, y, y', y'', ..., y(n)).

solvediffeq(F(x, y, y', y'', ..., y(n-1)), x, y, n, x0, x1, y0, y1, ..., yn-1): Повертає числові розв'язки диференціального рівняння y(n) = F(x, y, y', y'', ..., y(n-1)), y(x0) = y0, ..., y(n-1)(x0) = yn-1 у інтервалі x_0 <= x <= x1. Для числових розрахунків інтервал [x0, x1] розбивається на 1000 підінтервалів та MathTool повертає усі 1001 значення функції у відповідних точках.

Приклад: solvedeq(-y,t,y,2,0,10,1,0) повертає числові розв'язки диференціального рівняння y''(t) = -y(t), y(0) = 1, y'(0) = 0 for 0 <= t <= 10. Тут MathTool повертає 1001 значень y(t), оскільки приріст рівний 0.01.

table(F(x1, ..., xn)):Повертає таблицю усіх можливих значень логічного виразу F(x1, ..., xn), в залежності від значень змінних x1, ..., xn. Якщо n більше 20, то підрахунки проводитися не будуть.

tangent(F(x1, ..., xn), x1 = a1, ..., xn = an): Повертає рівняння дотичного простору до графііка F у точці (x1, ..., xn) = (a1, ..., an).

Приклад: tangent(x^3+x/2+3,x=2) поверне наступне рівняння:

Y=12+(25*(x-2))/2.


Додатково MathTool будує обидва графіки (функцій x^3+x/2+3 та 12+25*(x-2)/2).

taylordiffeq(F(x, y, y', y'', ..., y(n-1)), x, y, n, x0, y0, y1, ..., yn-1, m): Повертає поліном Тейлора степеня m розв'язку диференціального рівняння y(n) = F(x, y, y', y'', ..., y(n-1)), y(x0) = y_0, ..., y(n-1)(x0) = yn-1.

Приклад: taylordeq(-y,t,y,2,0,1,0,10) повертає поліном Тейлора степеня 10 розв'язку диференціального рівняння y''(t) = -y(t), y(0) = 1, y'(0) = 0. Точні розв'язки задається формулою y(t) = cos(t), тому послідовності степенів це

y(t) = cos(t) = 1 - t^2/2! + t^4/4! - t^6/6! + ...


Таким чином MathTool повертає

1 + t^4/24 + t^8/40320-(t^2/2 + t^6/720 + t^10/3628800).


undeffuncs(f1, ..., fn): Видаляє функції f1, ..., fn.

undefvars(x1, ..., xn): Змінні x1, ..., xn знову вільні змінні, тобто без присвоєних значень.

undefallfuncs(): Видаляє усі функції, що були присвоєні протягом сесії.

undefallvars(): Усі змінні, яким призначено значення протягом даної сесії, знову вільні змінні.

undefall(): Видаляє усі функції та змінні, що були присвоєні протягом сесії.

Усі консольні команди, що були запущені під час сесії зберігаються та можуть бути запущениі повторно використовуючи стілки (вверх = остання команда, вниз = наступна команда).
